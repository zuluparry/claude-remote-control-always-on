# Screenshots

Drop PNG screenshots here to be referenced from the main README.

**Suggested shots :**

- `phone-code-tab.png` — phone's Claude app Code tab showing "Command Centre" in the list (idle or attached)
- `phone-chat.png` — a conversation inside Command Centre demonstrating it responding (anything non-sensitive ; "list my Documents folder" is a safe demo prompt)
- `launchctl-output.png` — terminal screenshot of `launchctl list | grep command-centre` showing the PID
- `tmux-ls.png` — terminal screenshot of `tmux ls` showing `command-centre: 1 windows`

**Once added :** uncomment the `<!-- Screenshots -->` block in the main `README.md` to display them on the repo landing page.

**Recommended sizing :** PNG, under 500KB each, ~1200px wide for phone shots (iOS retina dimensions), ~1000px wide for terminal shots.

**Privacy check before pushing :** make sure screenshots don't show :
- Your real macOS username (blur or crop `/Users/yourname/` paths)
- Private session content
- Environment IDs you're actively using
- Any email addresses

Demo-grade content only.
